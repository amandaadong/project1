Writing function

Next, using the methods we used for Canada as a blueprint, I’ll create a function to get the data for all countries.

get_gdp_data <- function(country, year, key = my_key) {
  if (is.na(country) || is.na(year)) {
    return(NULL)
  }

  country <- str_replace_all(country, " ", "%20")
  base_url <- "https://api.api-ninjas.com/v1/gdp"
  url <- str_c(base_url, "?country=", country, "&year=", year)
  raw_data <- GET(url, add_headers("X-Api-Key" = my_key))
  json_data <- rawToChar(raw_data$content)
  tidy_data <- fromJSON(json_data)
  if (!is.data.frame(tidy_data) || nrow(tidy_data) == 0) {
    return(NULL)
  }
  if (!"gdp_per_capita_nominal" %in% names(tidy_data) || 
      is.na(tidy_data$gdp_per_capita_nominal)) {
    return(tibble(
      country = country,
      year = year,
      gdp_per_capita_nominal = NA
    ))
  }
  gdps <- tidy_data %>% select(country, year, gdp_per_capita_nominal)
}

Brief description of new dataset

We now have a new dataframe (which I’ve named “GDP_APIdata”) which contains 2194 rows with 4 variables. The 2194 rows correlate to the rows in “cases_year” (our original measles dataset) that have both country and year data. 3 of the variables in our new dataframe are country (as both full names and abbreviated) and year, while the last variable is “gdp_per_capita_nominal”. As the name suggests, this column contains the nominal GDP per capita for that country, in that year. For any country and year that did not have its nominal GDP per capita data available, their value is NA. All data was taken from API Ninja’s dataset.
